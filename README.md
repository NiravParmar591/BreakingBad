# Breaking Bad Fan Site

A cinematic fan website for Breaking Bad, built as a GSAP ScrollTrigger practice project. Features immersive scroll animations, character showcases, a Walt transformation timeline, and a seasons breakdown — all styled with a dark, gritty desert palette.

---

## Tech Stack

- **HTML5**
- **Tailwind CSS v4** — custom design tokens, utility classes
- **GSAP + ScrollTrigger + ScrollToPlugin** — all scroll animations
- **Vite** — dev server and bundler
- **Vanilla JS** — modular animation architecture

---

## Features

- **Cinematic Intro** — logo scale-in animation with scroll-triggered hero exit
- **Scrolling Marquee** — horizontal dialog text scrubbed by scroll
- **Character Showcases** — pinned scroll sections per character with staggered text reveals and grayscale toggle
- **Walt Transformation Timeline** — horizontal scroll section tracking Walter White's arc across 5 stages
- **Seasons Page** — animated season cards with notable episode tags
- **Character Quote Lines** — hover/tap to reveal iconic quotes per character
- **Animated Footer** — staggered entry with smooth scroll navigation
